# v1.6.0 — Eixo temporal, formatadores e escala plugável (P1, séries temporais A)

> **Origem:** improvements §6.1, §6.2, §6.4 · §15 passos 16–17 · roadmap Fase 3 · addons §1.3
> **Semver:** minor. Aditivo.
> **Risco:** médio.

## Objetivo

Transformar a biblioteca numa solução forte para séries temporais: eixo de
tempo, formatadores de eixo/tooltip e tratamento de dados ausentes. Deixar a
escala **plugável** (`linear`/`time`/`band`) para reaproveitar no bar chart.

## Escopo desta versão

### Escala plugável (addons §1.3) — fundação

Hoje `src/math/scale.ts` só faz `linear`. Tornar a escala plugável:

- [ ] `linear` — existente, vira uma das opções.
- [ ] `time` — destrava o eixo temporal desta versão.
- [ ] `band` — preparado agora; **destrava o bar chart na v1.9.0 quase de graça**.

> A escala serve **só** para time + band dentro de XY. Não é porta para pizza,
> radar ou mapas (addons §3).

### Eixo temporal (§6.1 · §15.16)

- [ ] Timestamps formatados corretamente; ticks mudam conforme zoom.
- [ ] Sem sobreposição excessiva; formatação customizada funciona.
- [ ] Domínio temporal suporta datasets grandes.

### Formatadores (§6.2 · §15.17)

- [ ] Eixo X e Y suportam formatador; série pode sobrescrever.
- [ ] Tooltip pode usar formatação diferente.
- [ ] Formatação não altera o valor numérico interno.

### Dados ausentes (§6.4)

- [ ] `NaN` não quebra escala; `break` cria interrupção visual.
- [ ] `connect` conecta extremos válidos; tooltip ignora ausentes.
- [ ] Stacking trata ausência de forma documentada.

## Pré-requisitos técnicos

- Regra documentada para `NaN` da [v1.3.0](./v1.3.0-contratos-dados.md) (§4.9)
  é a base do contrato de gaps aqui.
- **Preparar a v1.7.0:** ao mexer em domínio para o eixo temporal, planejar o
  curto-circuito de viewport em `updateGridDomain` (addons §2.1) — implementado
  na v1.7.0, mas o eixo temporal deve conviver com ele.

## Arquivos provavelmente afetados

- `src/math/scale.ts` — escala plugável `linear`/`time`/`band`.
- `src/math/ticks.ts` — geração de ticks temporais (segundo→mês).
- `src/math/format.ts` — formatadores de eixo/tooltip.
- `src/render/axes.ts` — render do eixo temporal.
- `src/charts/chart-base.ts` — gaps no domínio e no path.
- `src/types.ts` — tipo de escala, formatadores, política de gaps.
